# IELTS Writing Task 1: Academic Report Writing Master Guide

An elegant, comprehensive reference guide for IELTS Academic Writing Task 1, covering paragraph structures, report types, combinational charts, and non-negotiable strategic rules.

---

## 📊 1. Exam Structure & Paragraph Layout

| Parameter | Standard Requirements |
| :--- | :--- |
| **Target Word Count** | Minimum **150 Words** |
| **Time Allocation** | **20 Minutes** (Recommended) |
| **Nature of Task** | Data/Process Objective Report Writing |

### 🧱 Standard Paragraph Breakdown
To maximize Coherence and Cohesion (CC), your report must rigorously follow this 4-paragraph layout:

1. **Paragraph 1: Introduction**
   * **Length:** Exactly 1 Sentence.
   * **Strategy:** Paraphrase the prompt question using synonyms and alternative structures. Do not add anything extra.
2. **Paragraph 2: Overview**
   * **Length:** Exactly 2 Sentences.
   * **Strategy:** Highlight the overall main trends, highest/lowest points, or key stages. No specific numbers/data points should be placed here.
3. **Paragraph 3: Main Body Paragraph 1 (Details)**
   * **Length:** 3 to 4 Sentences.
   * **Strategy:** Group and describe relevant specific data or features logically.
4. **Paragraph 4: Main Body Paragraph 2 (Details)**
   * **Length:** 3 to 4 Sentences.
   * **Strategy:** Compare or describe the remaining features/segments of the graph.

## 📋 2. Core Types of Reports

IELTS Task 1 charts generally fall into these 7 distinct categories:

1. **Line Chart**
2. **Bar Chart**
3. **Pie Chart**
4. **Table**
5. **Process Diagram** (Flow Charts)
6. **Maps** (Geographical layout changes over time)
7. **Combined Diagrams** (Mixed Charts containing multiple data types)

### 🔀 Combined Diagram Permutations List
When dealing with Combined/Mixed graphs, they typically form the following pairs:
* Line Chart + Line Chart
* Line Chart + Bar Chart
* Line Chart + Pie Chart
* Line Chart + Table
* Bar Chart + Pie Chart
* Bar Chart + Table
* Bar Chart + Bar Chart
* Table + Pie Chart
* Table + Table

## 🔑 3. Essential Rules & Practical Strategies (Notes 1–12)

#### 🛑 Rule 1: Do Not Over-Detail
* **Action:** Avoid dumping every single minor detail into your Main Body paragraphs. Select key points, transitions, and trends to maintain analytical clarity.

#### 🧠 Rule 2: Never Give Opinions
* **Action:** This is a strict objective report. Never analyze *why* a trend happened or include personal interpretations or assumptions. Write only what is visually true in the graph.

#### ⏳ Rule 3: Grammatical Tense Accuracy
* **Action:** Always look at the years/time frames provided in the prompt. Use the appropriate tense (Past, Present, or Future) dynamically based on the timeline given.

#### 🏢 Rule 4: Stick to the Mandatory Paragraph Structure
* **Action:** You **must** structure your report precisely using the established flow: **Introduction ➡️ Overview ➡️ Main Body 1 ➡️ Main Body 2**.

#### 📍 Rule 5: Professional Preposition Usage
* **Action:** Be highly precise with prepositions when quoting numerical metrics and data movements.
  * *Examples:* 
    * "Increased **by** 10%" (Margin of change)
    * "Stood **at** 50,000" (Exact value)
    * "Rose from X **to** Y" (Starting and ending boundaries)

#### 📝 Rule 6: Paraphrasing the Introduction
* **Action:** For the first paragraph, paraphrase the given question statement using your own words. **Aim to rewrite around 45% of the original text** using synonyms, and do not include any external information.

#### 📉 Rule 7: Writing the Overview without Numbers
* **Action:** The Overview sentence block must **never** contain specific numbers, data values, raw figures, or precise percentages. You may only state the general year or timeline if necessary to mark the trend (e.g., *"...over the 10-year period"*).

#### 📈 Rule 8: Main Body Strategy for Constant/Continuous Trends
* **Action:** If a Line Chart or Bar Chart shows a continuous increase or decrease across the entire timeline, your Main Bodies should highlight the **starting point (beginning)**, the **intermediary key movements (middle shifts)**, and the final state at the **end**.

#### 📊 Rule 9: Main Body Strategy for Segmented Trends
* **Action:** If a Line Chart or Bar Chart shows erratic patterns (fluctuating rises and falls), you must break the graph down into segments. Analyze and explain **each distinct segment step-by-step** across your Main Body paragraphs.

#### ⚙️ Rule 10: Passive Voice for Processes and Maps
* **Action:** For **Process Diagrams** and **Maps**, always use the **Passive Voice** structure.
  * *Formula:* $\text{Be Verb} + \text{Main Verb (3rd form/Past Participle)}$
  * *Example:* "The wheat *is harvested* and then *transported* to the factory."

#### 🔮 Rule 11: Simple Tenses for Timelines
* **Action:** When evaluating past, present, and future data blocks, stick to the **Simple Tense variants** (Simple Past, Simple Present, or Simple Future).

#### ❌ Rule 12: No Conclusion Needed
* **Action:** Task 1 requires an *Overview*, **NOT a Conclusion**. There is no room for creative wrap-ups or summarizing conclusions since you are reporting facts, not arguing a point.
